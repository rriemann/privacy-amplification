# About this Software

This software was developed as part of my master thesis:
  "Implementation Of A Control Unit For A Quantum Key Distribution Experiment Including Postprocessing"

Submitted:
  Berlin, 1st of February 2013

# Purpose

This software does the post processing for raw quantum key measurement data including:

  - sifting
  - error estimation
  - error correction
  - privacy amplification
  - token based authentication
